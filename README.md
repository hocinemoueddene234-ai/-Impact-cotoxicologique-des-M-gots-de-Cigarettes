# -Impact-cotoxicologique-des-M-gots-de-Cigarettes
Ce projet se concentre sur les déchets les plus visibles et polluants liés à la nicotine : les mégots.


![Sans titre](https://github.com/user-attachments/assets/65c3fcc4-cf80-491e-be93-78895ae68474)


Substance chimique 	Effets écotoxicologiques
Nicotine	Toxique pour la vie aquatique, pouvant entraîner la mort des poissons et des mollusques.
Arsenic, Mercure, Plomb	Métaux lourds toxiques qui s'accumulent dans les chaînes alimentaires.
Acide cyanhydrique 	Poison violent.
Naphtalène	Polluant organique toxique.
Ammoniac	Toxique pour la vie aquatique.
